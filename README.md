# 什么云科技有限公司官网

专业的数字化解决方案提供商官网，基于 Vue 3 + Vite 构建。

## 项目特性

- ⚡️ **高性能** - 基于 Vite 构建，开发和生产环境都极速
- 🌍 **国际化** - 支持中英文切换，自动检测浏览器语言
- 📱 **响应式设计** - 完美适配各种设备（桌面端、平板、移动端）
- 🎨 **现代化UI** - 精美的渐变设计和流畅的动画效果
- 🚀 **工程化** - 清晰的目录结构，易于维护和扩展
- 🎯 **性能优化** - 全面的性能优化，确保流畅的用户体验
- ✨ **动画效果** - 科技感背景动画，粒子效果和渐变光球

## 技术栈

- **Vue 3.5** - 渐进式 JavaScript 框架，使用 Composition API
- **Vue Router 4** - 官方路由管理器，支持路由懒加载
- **Vite 7** - 下一代前端构建工具，极速的开发体验
- **Terser** - JavaScript 压缩工具，优化生产构建

## 项目结构

```
src/
├── components/              # 公共组件
│   ├── TheHeader.vue        # 头部导航组件（固定顶部，支持滚动效果）
│   ├── TheFooter.vue        # 页脚组件（包含备案信息）
│   └── TechBackground.vue   # 科技背景组件（粒子动画、渐变光球）
├── views/                  # 页面组件
│   ├── HomeView.vue        # 首页（Hero区域、服务介绍、优势、流程、技术栈、统计数据）
│   ├── AboutView.vue        # 关于我们（公司介绍、使命愿景、发展历程、团队）
│   ├── ServicesView.vue     # 服务项目（详细服务介绍）
│   ├── CasesView.vue        # 成功案例（项目案例展示）
│   └── ContactView.vue     # 联系我们（联系表单和联系信息）
├── composables/            # 组合式函数
│   └── useI18n.js          # 国际化处理（中英文切换、语言检测）
├── router/                 # 路由配置
│   └── index.js            # 路由定义（支持懒加载）
├── App.vue                 # 根组件
└── main.js                 # 入口文件
```

## 快速开始

### 环境要求

- Node.js: `^20.19.0 || >=22.12.0`
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发

```bash
# 启动开发服务器（默认端口 5173）
npm run dev
```

开发服务器启动后，访问 `http://localhost:5173` 即可查看网站。

### 构建

```bash
# 构建生产版本
npm run build
```

构建产物将输出到 `dist/` 目录。

### 预览

```bash
# 预览生产构建
npm run preview
```

## 功能说明

### 页面路由

- `/` - **首页**：展示公司介绍、核心服务、技术优势、服务流程、技术栈和统计数据
- `/about` - **关于我们**：公司使命、愿景、发展历程和核心团队介绍
- `/services` - **服务项目**：详细的服务介绍（小程序开发、AIOT、Web开发、云服务等）
- `/cases` - **成功案例**：展示成功项目案例和成果
- `/contact` - **联系我们**：联系表单和公司联系信息

### 国际化

项目支持中英文切换：

- **自动检测**：首次访问时根据浏览器语言自动选择
- **手动切换**：点击导航栏的语言切换按钮
- **持久化**：手动切换的语言设置会保存在 localStorage 中
- **自动检测不保存**：自动检测的语言不会保存，下次访问仍会根据浏览器语言选择

### 性能优化

项目进行了全面的性能优化，确保流畅的用户体验：

#### 1. 动画性能优化
- 使用 `transform3d` 和 `will-change` 启用 GPU 加速
- 减少粒子数量（从50个优化到30个）
- 预计算动画样式，避免重复计算
- 使用 `contain` 属性限制重绘范围

#### 2. 滚动性能优化
- 滚动事件使用 `requestAnimationFrame` 节流
- 添加 `passive: true` 提升滚动性能
- 优化 IntersectionObserver 配置

#### 3. 构建优化
- **代码分割**：Vue、Vue Router 和其他依赖分离打包
- **CSS 代码分割**：按需加载样式
- **Terser 压缩**：多轮压缩，移除 console 和 debugger
- **依赖预构建**：优化开发时的依赖加载

#### 4. 资源优化
- 路由懒加载，减少初始加载体积
- 优化动画关键帧，使用 GPU 友好属性
- 减少不必要的重新渲染

## 开发规范

### 代码风格

- 使用 Vue 3 Composition API
- 组件使用 `<script setup>` 语法
- 样式使用 scoped CSS
- 使用 TypeScript 风格的 JSDoc 注释（可选）

### 命名规范

- 组件文件：PascalCase（如 `TheHeader.vue`）
- 组合式函数：camelCase（如 `useI18n.js`）
- CSS 类名：kebab-case（如 `header-content`）

## 部署

### 构建生产版本

```bash
npm run build
```

### 部署到静态托管

构建完成后，将 `dist/` 目录的内容部署到任何静态文件服务器即可：

- **Vercel**：直接连接 GitHub 仓库，自动部署
- **Netlify**：拖拽 `dist/` 目录或连接 Git 仓库
- **GitHub Pages**：使用 GitHub Actions 自动部署
- **Nginx**：将 `dist/` 目录内容复制到 Nginx 的静态文件目录

### 环境变量

项目使用 Vite 的环境变量系统，可以在 `.env` 文件中配置：

- `.env` - 所有环境的默认变量
- `.env.local` - 本地环境变量（会被 git 忽略）
- `.env.production` - 生产环境变量
