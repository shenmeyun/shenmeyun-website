# 什么云科技有限公司官网

专业的数字化解决方案提供商官网，基于 Vue 3 + Vite 构建。

## 技术栈

- Vue 3.5
- Vue Router 4
- Vite 7

## 快速开始

### 环境要求

Node.js: `^20.19.0 || >=22.12.0`

### 安装依赖

```bash
npm install
```

### 配置环境变量

```bash
cp .env.example .env
```

根据需要修改 `.env` 文件中的配置。

### 开发

```bash
npm run dev
```

访问 `http://localhost:5173`

### 构建

```bash
npm run build
```

### 预览

```bash
npm run preview
```

### 部署

```bash
npm run deploy
```

部署到 `shenmeyun.github.io` 仓库的 `master` 分支。

**首次部署前：**
- 确保 `shenmeyun.github.io` 仓库已创建
- 在 GitHub Pages 设置中选择 `master` 分支作为 Source
